This project presents an interactive Pizza Sales Dashboard developed using Power BI to analyze sales performance and customer ordering patterns.

## Dashboard Highlights

Total revenue and total orders overview

Best-selling pizza categories

Monthly order trends

Sales distribution by pizza size

Revenue analysis by pizza type

Interactive filters by month for detailed exploration

## Tools Used

Power BI

DAX Measures

Data Modeling

Data Visualization

## Objective 

The objective of this dashboard is to analyze pizza sales data and provide clear insights into revenue drivers, popular products, and seasonal trends to support data-driven business decisions.
